# Diplom_1
Unit-тесты для проекта Stellar-Burgers
---
Technology stack:
Java 11,
Junit 4.13.1,
Mockito 3.12.4,
Maven 3.8.1,
Jacoco 0.8.10,
Byte-buddy 1.11.16

Запуск отчета Jacoco для оценки покрытия кода выполняется командой "mvn verify".
Отчет находится в папке target/site/jacoco/index.html запускается в браузере.
